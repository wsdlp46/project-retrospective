# project-retrospective

项目复盘 Skill，在完成一个项目或任务后系统复盘智能体和知识库的表现。

## 功能

- 识别提示词缺陷、知识库缺失和可技能化的重复流程
- 驱动五级联动修复：提示词 → 知识库 → 技能 → 案例 → 规范
- 输出归因表、技能候选评估、变更记录

## 触发

说"复盘""回顾项目""优化智能体""整理经验""改进系统"等。

## 安装

```bash
cp -r project-retrospective ~/skills/
```
